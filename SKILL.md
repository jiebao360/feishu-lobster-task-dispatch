## 📋 技能描述

通过**飞书群聊 IM 消息**向龙虾机器人分配任务，实现公开透明的任务编排和追踪。

**核心能力**:
- ✅ 通过 `feishu_im_user_message` 工具发送任务通知到**群聊**
- ✅ **@提及相关机器人**，确保它们收到通知
- ✅ 所有龙虾在群内可见任务，公开确认和协作
- ✅ 支持动态扩展龙虾数量和职能
- ✅ 自动解析文档链接，智能拆解任务
- ✅ 创建飞书多维表格追踪进度
- ✅ 汇总所有成果到飞书文档

**适用场景**:
- 用户说"分享 XXX 主题"时自动触发
- 批量任务分配给多个职能虾
- 需要群内公开确认和协作的任务
- 跨机器人协作项目
- 复杂任务拆解和分配

## 🔄 完整执行流程

### Step 1: 解析任务需求
```python
# 读取用户分享的文档链接
doc_content = feishu_fetch_doc(doc_id="https://xxx.feishu.cn/wiki/xxx")

# 解析文档内容，提取核心主题和关键信息
theme = extract_theme(doc_content)
key_points = extract_key_points(doc_content)

# 根据主题拆解为 N 个子任务
tasks = decompose_tasks(theme, key_points)

# 匹配对应的龙虾职能
lobster_mapping = match_lobster_tasks(tasks)
```

### Step 2: 发送群通知（关键步骤）✅
```python
# 通过 feishu_im_user_message 工具发送到飞书群聊
feishu_im_user_message(
    action="send",
    receive_id="oc_8faef5990acd30c853ad72f01fcb8f09",
    receive_id_type="chat_id",  # 必须是群聊，不能是私信
    msg_type="text",
    content='{"text":"🦞 任务分配通知\\n\\n@飞书工作助手 @第二大脑笔记助手..."}'
)
```

**关键点**:
- ✅ 必须发送到**群聊** (chat_id)，不能是私信
- ✅ 必须**@提及相关机器人**，确保它们收到通知
- ✅ 所有龙虾在群内可见任务，公开确认和协作
- ✅ 支持动态扩展龙虾数量

### Step 3: 等待各龙虾确认
- 各龙虾在群内回复确认
- 格式：✅ 龙虾 X 收到任务 LOBSTER-XX
- 记录确认时间和预计完成时间

### Step 4: 龙虾执行任务
- 各龙虾根据任务要求执行
- 完成后在群内汇报结果
- 主代理收集所有完成汇报

### Step 5: 创建任务追踪表
```python
# 创建飞书多维表格
bitable = feishu_bitable_app(action="create", name="🦞 龙虾任务追踪表 - 日期")

# 添加字段

    {"field_name": "任务编号", "type": 1},  # 文本

    {"field_name": "任务内容", "type": 1},
    {"field_name": "状态", "type": 3},  # 单选
    {"field_name": "开始时间", "type": 5},  # 日期
    {"field_name": "完成时间", "type": 5},
    {"field_name": "输出物", "type": 15},  # 超链接
    {"field_name": "备注", "type": 1}
]

# 批量创建 N 条记录
records = create_records(tasks, lobster_mapping)
```

### Step 6: 汇总成果
- 各龙虾完成后在群内汇报
- 负责汇总的龙虾创建汇总飞书文档
- 更新多维表格为已完成
